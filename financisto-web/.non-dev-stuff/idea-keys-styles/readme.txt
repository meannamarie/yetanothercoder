Prior Requirement:

   1. Download and install IdeaKeyScheme written by Santosh
   2. Restart Eclipse.

Preferences (Default Idea Java Code Style, Syntax color, Comment/Javadoc format, etc)

   1. Download Idea-Eclipse-Prefs.epf
   2. Import this preferences file into your Eclipse (File -> Import -> Preferences (Under General category) -> Specify path to the downloaded file -> Select 'Import All' - > Click finish)
   3. Restart Eclipse.
